# Expense Manager

Web application for expense management built with HTML, CSS, and Vanilla JavaScript.

---

# Current Project State

The application currently includes:

## Core
- Expense CRUD
- Data persistence
- Dynamic calendar
- Month navigation
- Active day selection
- Automatic categories
- Monthly statistics
- Section navigation
- Modular rendering

---

# Current Architecture

## Home
Main section focused on daily usage.

### Currently contains:
- Monthly calendar
- Active day selection
- Expense list for the selected day
- Dynamic expense form

---

## Statistics
Section focused on data interpretation and visualization.

### Currently contains:
- Daily total
- Monthly total
- Monthly categories
- Category percentages
- Dominant category
- Highest monthly expense
- Daily average
- Dynamic bar charts

### Future goals:
- Spending trends
- Small expense detection
- Visual comparisons
- Advanced charts
- Financial insights
- Dynamic category colors

---

## Summaries
Section focused on time-based summaries and history.

### Planned features:
- Weekly summary
- Monthly summary
- Yearly summary
- Month-to-month comparison
- Selected month history

### Navigation concept:
This section will not use a calendar.

Instead, it will use tab/view navigation:

- Weekly
- Monthly
- Yearly
- Comparison
- History
